# nba-final-project

## Project Description
This project analyzes NBA team performance using data pulled from the basketball reference. 
The goal of the project is to collect, clean, and analyze game-level data for selected NBA teams and 
present insights using Python and data analysis techniques.

## Data Source
- basketball reference
- NBA team game results from the 2019–2020 season

## Planned Features
- Retrieve NBA game data using `LeagueGameFinder`
- Filter and clean data using pandas
- Combine multiple team datasets into a single DataFrame
- Export cleaned data to CSV
- Perform exploratory data analysis and visualizations

## Tools & Technologies
- Python
- pandas
- Jupyter Notebook / VS Code
- GitHub
